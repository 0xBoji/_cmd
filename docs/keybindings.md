# Keybindings

## CLI

| Action | Shortcut |
| :--- | :--- |
| Quit | `q`, `Ctrl+C` |
| Select next agent | `Down`, `j` |
| Select previous agent | `Up`, `k` |
| Page down | `PageDown` |
| Page up | `PageUp` |
| Select first agent | `Home` |
| Select last agent | `End` |
| Toggle grid/focus mode | `Tab` |
| Cycle filter mode | `f` |
| Start search | `/` |
| Clear search | `Esc` |
| Finish search | `Enter` |
| Delete search character | `Backspace` |

## Desktop

| Action | Shortcut |
| :--- | :--- |
| Select next agent | `Down`, `j` |
| Select previous agent | `Up`, `k` |
| Page down | `PageDown` |
| Page up | `PageUp` |
| Cycle filter mode | `f` |
| New terminal tab | `Cmd+T` |
| Close terminal tab | `Cmd+W` |
| Select terminal 1-9 | `Cmd+1` through `Cmd+9` |
| Next terminal tab | `Cmd+Right` |
| Previous terminal tab | `Cmd+Left` |
| Toggle grid/focus mode | `Cmd+G` |
| Open terminal transcript search | `Cmd+F` |
